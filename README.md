# 🔱 ঐতিহাসিক চড়ক পূজা ও চড়কবাড়ি (বোঁথড় গ্রাম)

একটি শতাব্দী প্রাচীন আধ্যাত্মিক ঐতিহ্য এবং বংশপরম্পরায় চলে আসা পবিত্র চড়ক উৎসবের ডিজিটাল মহাফেজখানা। এই ওয়েব অ্যাপ্লিকেশনটি বোঁথড় গ্রামের চড়কবাড়ির গৌরবময় ইতিবৃত্ত, সন্ন্যাসীদের বংশধারা এবং শাস্ত্রীয় আচারসমূহকে বিশ্বের দরবারে তুলিয়া ধরিবার একটি ক্ষুদ্র প্রয়াস।

---

## 📖 মূল বৈশিষ্ট্যসমূহ (Core Features)

* **অমর ইতিবৃত্ত (Ancient History):** পাণ্ডুলিপি হইতে সংগৃহীত সপ্তশত বর্ষের পুরাতন ইতিহাস।
* **বংশধারা (Monk Lineage):** প্রধান ১৩ বিট সন্ন্যাসী, সহকারী এবং আগত সন্ন্যাসীগণের বিস্তারিত তালিকা।
* **সংকল্প ও সেবা (Oath & Services):** ভক্তগণের তরে মানত ও শাস্ত্রীয় সেবার নির্দেশিকা।
* **আধ্যাত্মিক প্রচ্ছদ (Heroic Entrance):** শতাব্দী প্রাচীন ঐতিহ্যের এক মহিমান্বিত উপস্থাপনা।
* **শাস্ত্রীয় আচার (Rituals):** উৎসবের প্রতিটি পবিত্র কার্যের সময়সূচী ও বর্ণনা।

## 🛠 প্রযুক্তিগত কাঠামো (Tech Stack)

এই প্রকল্পটি আধুনিক ওয়েব প্রযুক্তির সমন্বয়ে নির্মিত হইয়াছে:

* **Framework:** [Next.js 14/15](https://nextjs.org/) (App Router)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Animations:** Framer Motion / CSS Keyframes
